# Global-Electronics-Sales-Analysis
Global Electronics Sales Analysis – SQL &amp; EDA A collection of SQL queries and exploratory data analysis notebooks to analyze sales, products, stores, customers, and revenue trends for the Global Electronics dataset.
This repository contains a structured analysis of the Global Electronics Sales dataset using SQL queries and Python-based exploratory data analysis (EDA). The goal is to extract key business insights and support data-driven decision-making.

Key Features:

Sales Analysis: Total revenue, monthly trends, and seasonality.

Product Insights: Top-selling products, categories, and quantity sold.

Store Performance: Revenue by store, city, country, and continent.

Customer Analysis: Segmentation, average order value, and demographic insights.

Currency Impact: Revenue analysis across different currencies.

Category Trends: Monthly revenue trends per product category.

Repository Structure:

sql_queries/ → Contains 10 SQL queries addressing critical business questions.

notebooks/ → Python Jupyter notebooks for visualizing insights and performing EDA.

data/ → Optional folder for dataset CSV/Excel files.

README.md → Project overview, instructions, and insights.

Objective:
To demonstrate practical SQL skills, data analysis, and visualization for a retail/e-commerce dataset, producing actionable business insights.
